# AthlynXAI Doctrine

The doctrine layer is the *why* of AthlynXAI. Every product surface, brand surface, and growth surface ships under one or more of these documents. Agents may draft and recommend; only Chad approves shipping.

## The doctrine set

| Doctrine | What it governs | Quick rule |
|---|---|---|
| [Founding Manifesto](./FOUNDING_MANIFESTO.md) | Why AthlynXAI exists. The mission frame. | If it doesn't serve an athlete, it doesn't belong. |
| [Small Circle Doctrine](./SMALL_CIRCLE_DOCTRINE.md) | Who's inside, who's trusted, how trust is earned. | Keep the circle small until the platform can carry the weight. |
| [Brand Doctrine](./BRAND_DOCTRINE.md) | How AthlynXAI looks, sounds, and feels. | Cyan, calm, premium. No buzzword salad. |
| [Smith Standard](./SMITH_STANDARD.md) | The competitive creative standard for every shipped surface. Includes the Maddux Sequence. | Forge it. Sharpen it. Test it. Make it useful before it reaches a user. |

## How to read these together

- **Founding Manifesto** sets the mission.
- **Small Circle** sets the trust model.
- **Brand Doctrine** sets the visual + voice rules.
- **Smith Standard** sets the *competitive bar* and the *Maddux Sequence* that governs every external action — product, growth, capital.

Every PR that touches a user-facing surface must answer:

1. Which pillars of the Smith Standard does this satisfy?
2. Which Maddux pitch is this — Setup (1), Spot (2), Change of pace (3), or Put-away (4)?
3. Does this honor Small Circle, Brand, and Founding Manifesto rules?

If you can't answer all three, it doesn't ship.
