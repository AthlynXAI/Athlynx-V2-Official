# AthlynXAI OS Auto SEO Research Pack

This folder supports the Auto SEO roadmap in `docs/specs/athlynxai-os-auto-seo-roadmap.md`.

The module is intended to become the SEO, content, backlink, and CRM growth layer inside AthlynXAI OS. It is research and architecture only at this stage. It does not publish content, send outreach, touch secrets, run database writes, or alter production behavior.

| File | Purpose |
|---|---|
| `auto-seo-market-research-findings.md` | NIL, athlete branding, recruiting, and deal-flow source findings. |
| `auto-seo-competitor-summary.md` | Readable competitor traffic summary generated from SimilarWeb data. |
| `auto-seo-competitor-summary.csv` | Supporting structured competitor summary for later analysis. |
