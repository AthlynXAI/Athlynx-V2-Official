# Logo and Brand Hierarchy Gate

Production brand assets must use local, company-approved marks only. Do not hotlink third-party sports-logo CDNs, remote avatar generators, or Gravatar identicons for public production UI. NIL vault artwork, investor/private vault files, and raw archive assets do not belong in public static directories.

DHG ecosystem pages may use the approved DHG ecosystem mark. Dedicated subsidiary pages must use their own approved company-specific logos when available. If a logo is not approved or not present, use the local AthlynX fallback mark and label the area clearly rather than introducing placeholder or third-party imagery.

Every logo/asset cleanup is checked off only after local validation, GitHub push, Vercel READY deployment, and live route verification.
