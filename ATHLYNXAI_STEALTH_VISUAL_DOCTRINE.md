# AthlynXAI Permanent Stealth Visual Doctrine

**Owner:** Chad A. Dozier  
**Status:** Permanent operating rule  
**Scope:** AthlynXAI OS, public website, app surfaces, sport pages, athlete profiles, social image cards, decks, documents, ads, and all public-facing creative.

## Core rule

AthlynXAI must present in **stealth mode**. The visual system is built on metallic black, deep navy, dark blue, graphite, and controlled cyan or ice-blue accents. The brand should look clean, premium, serious, and professional.

AthlynXAI is **the X of the industry**. The X-Factor logo/icon standard is a stealth black foundation with a clean white or ice-white X mark, supported only by controlled cyan/ice-blue accents when needed. This identity is locked in `ATHLYNXAI_X_FACTOR_IDENTITY_DOCTRINE.md`.

## Permanent visual standards

| Category | Permanent rule |
|---|---|
| Base palette | Metallic black, graphite, near-black, deep navy, and dark blue. |
| Accent palette | Controlled cyan, ice-blue, electric blue, or subtle steel-blue accents only. |
| Prohibited colors | No gold, no yellow, no rainbow gradients, no rainbow/multicolor styling, and no bright novelty palettes. |
| Emoji usage | No emojis unless truly necessary for platform function or explicitly approved by the owner. |
| Social cards | Original, mobile-readable, high-contrast, dark/stealth design. No raw screenshots, no copied creative, no placeholder art. |
| X-Factor mark | Use the true AthlynXAI X-Factor identity direction: metallic black or black field, clean white/ice-white X mark, premium industry-leading presentation. |
| Website and app UI | Same stealth foundation across all sports, app surfaces, public pages, and profile experiences. |
| Exceptions | Any exception to color or emoji rules requires explicit owner approval for that specific asset or page. |

## Approved creative direction

The approved direction is **metallic black stealth mode** with dark navy structure and precise cyan/ice-blue highlights. Designs should feel like a premium sports-intelligence operating system, not a loud consumer template.

## Implementation requirement

All future design work, generated graphics, social media assets, sport pages, profile surfaces, app icons, logos, and public website changes must check this doctrine before release. If a page or asset uses gold, yellow, rainbow colors, unnecessary emojis, non-stealth styling, or an incorrect/placeholder X-Factor mark, it must be treated as failing the AthlynXAI OS visual standard until corrected.
